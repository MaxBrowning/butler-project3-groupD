OUR PROJECT

Predicting Property Value with Machine Learning

Using data collected from Indiana Stats for analysis, a model was created using machine learning which allowed property value prediction. An app was created and deployed to allow user input to predict a propertys value. Along side of the application, a dashboard was created to allow visualization of the data. Data filtering and comparables were also created.



For this project we used:
Python | HTML | CSS |SQL | Flask | API | SKLearn | Machine Learning
Microsoft Excel | SQLAlchemy | Pandas | Jupyter Notebook | Matplotlib
